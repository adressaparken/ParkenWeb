API Requests

Request Example using pagination 
GET http://localhost:10010/sensorDataList?sensor=CO2&limit=4&offset=100
Parameters
Name Located in Description Required Type
sensor query The sensor name No string
limit query The limit of records Yes number
offset query The offset of records Yes number
Request Example by dates 
GET http://localhost:10010/sensorDataByDate?start_date=2017-05-29 15:16:28&end_date=2017-07-29 19:17:58
Parameters
Name Located in Description Required Type
sensor query The sensor name No string
start_date query The start date Yes string
end_date query The end date Yes string
