API Requests

Example using pagination 
GET http://parken.perseum.com:10010/v1/sensorDataList?sensor=CO2&limit=10&offset=100

This API method returns the latest number of results (limit) withing a sertain offset.

If the limit=100 and offset=0 it will return the last 100 results. Changing the offset=1 it will bring the next 100 results and so on.

If a sensor is specified it will return results just for that sensor, otherwise it will return the last results independently of the sensor.

Parameters
Name Located in Description Required Type Example
sensor query The sensor name No string CO2,HUMA,LUM,MCP,PIR,TCA
limit query The limit of records Yes number 100
offset query The offset of records Yes number 1
Request Example by dates 
GET http://parken.perseum.com:10010/v1/sensorDataByDate?start_date=2016-12-07%2015:57:23&end_date=2017-05-13%2015:57:23&interval=MONTH

This API method returns results between to dates and groups them by an interval (YEAR,MONTH,WEEK,DAY,HOUR).

This can be done also for a specific sensor.

Parameters
Name Located in Description Required Type Example
sensor query The sensor name No string CO2,HUMA,LUM,MCP,PIR,TCA
start_date query The start date Yes string 2017-05-29 15:16:28
end_date query The end date Yes string 2017-06-29 15:16:28
interval query The data interval Yes string YEAR,MONTH,WEEK,DAY,HOUR
